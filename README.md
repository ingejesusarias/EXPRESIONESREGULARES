# EXPRESIONESREGULARES
ANALIZADOR DE EXPRESIONES EJECUTABLES 
este proyecto es una aplicación de Java Swing que muestra una interfaz gráfica de usuario
el cual es un analizador léxico. 
Esta interfaz le dara al usuario un campo donde ingresara una expresión y luego la analizará
Los resultados del análisis se mostraran  en dos áreas de texto en la misma ventana en la ventana.
